ท่านสามารถดาวน์โหลดใบรับรองอิเล็กทรอนิกส์ได้ที่ ดาวน์โหลดใบรับรองฯ และท่านดำเนินการจัดเก็บไฟล์ใบรับรองอิเล็กทรอนิกส์ และ Private Key ได้ตามขั้นตอนดังนี้

ขั้นตอนการจัดเก็บไฟล์ใบรับรองอิเล็กทรอนิกส์ และ Private Key

1. เปิดโปรแกรม OpenSSL และคัดลอกคำสั่ง

pkcs12 -export -in cert.cer -inkey cert.key -out certificate.p12

และวางลงไปในหน้าต่างโปรแกรม OpenSSL จากนั้นกดปุ่ม Enter

คำอธิบายคำสั่ง

-export : สร้างไฟล์ .p12

-in cert.cer : ไฟล์ใบรับรองอิเล็กทรอนิกส์ (.cer)

-inkey cert.key : ไฟล์ Private Key (.key)

-out certificate.p12 : ไฟล์ .p12 ที่จะสร้าง

2. ตั้งรหัสผ่านและยืนยันในโปรแกรม OpenSSL (ตัวอักษรจะไม่แสดงขณะพิมพ์)

3. นำไฟล์ใบรับรองฯ (certificate.p12) ไปใช้งาน

กรณีต้องการจัดเก็บใบรับรองฯ ลงในอุปกรณ์ eToken (P11) ให้ผู้ใช้ดำเนินการต่อจากขั้นตอนที่ 3 ดังนี้

⏵ เปิดโปรแกรม Safenet Authentication Client Tools (SAC)
- สำหรับ Windows 32-bit
- สำหรับ Windows 64-bit
⏵ คลิกที่ปุ่ม Advanced View (ไอคอนรูปเฟือง) มุมขวาบน
⏵ คลิกขวาที่ชื่ออุปกรณ์ eToken > เลือก "Import Certificate" > ใส่รหัส eToken
⏵ เลือก "Import a certificate from a file" > กดปุ่ม OK > เลือกไฟล์ใบรับรองฯ (.p12) ที่ต้องการนำเข้า
⏵ ใส่รหัสของไฟล์ใบรับรองฯ (.p12) > กดปุ่ม OK > โปรแกรมแสดงการนำเข้าใบรับรองฯ สำเร็จ
